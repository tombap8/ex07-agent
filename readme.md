# Aura Calc - Modern Web Calculator

Aura Calc는 글래스모피즘(Glassmorphism) 스타일의 유려한 사용자 인터페이스와 편리한 부가 기능을 제공하는 반응형 웹 계산기 프로그램입니다.

## 🌟 주요 특징

- **글래스모피즘 디자인:** 아름다운 배경 그라데이션 오브(Orbs)와 반투명 블러 효과를 적용해 깊이감 있고 세련된 UI를 제공합니다.
- **다크/라이트 테마:** 사용자의 기호에 맞게 다크 모드와 라이트 모드 간에 테마를 실시간으로 전환할 수 있습니다. (선택한 테마는 브라우저 저장소에 영구 보존됩니다)
- **계산 기록 관리:** 이전 계산 내역을 기억하고 필요할 때 불러와 수식에 재활용할 수 있는 히스토리 패널 기능이 포함되어 있습니다. (`localStorage`를 통해 페이지를 새로고침해도 유지됨)
- **전체 키보드 매핑:** 숫자 키패드 및 기본 산술 연산자(`+`, `-`, `*`, `/`), 실행(`Enter` / `=`), 지우기(`Backspace`), 초기화(`Esc`) 등 데스크톱 키보드 조작을 완벽히 지원합니다.
- **부드러운 미세 애니메이션:** 키 입력 피드백 및 패널 전환 애니메이션을 지원하여 높은 조작 일체감을 줍니다.

## 🚀 실행 및 테스트 방법

이 웹 어플리케이션은 순수 정적 파일(Vanilla HTML/CSS/JS)로 작성되어 특정한 빌드 단계나 패키지 설치 없이 즉시 실행할 수 있습니다.

### 방법 1: 로컬 브라우저에서 바로 열기
1. `index.html` 파일을 더블 클릭하여 기본 웹 브라우저(Chrome, Edge 등)로 실행합니다.

### 방법 2: 로컬 웹 서버로 열기 (권한 및 LocalStorage가 더 완벽히 작동하는 개발 환경 권장)
1. 프로젝트 폴더 터미널에서 아래와 같이 초경량 로컬 웹 서버를 실행합니다.
   ```powershell
   # Python이 설치된 경우
   python -m http.server 8000
   ```
2. 웹 브라우저를 열고 `http://localhost:8000`에 접속합니다.

## ⌨️ 키보드 단축키 가이드

마우스 조작 외에도 키보드로 즉시 계산기를 사용할 수 있습니다:

| 동작 | 키보드 키 | 설명 |
| :--- | :--- | :--- |
| **숫자 입력** | `0` ~ `9`, `.` | 화면에 숫자 및 소수점을 입력합니다. |
| **연산자 입력** | `+`, `-`, `*` (×), `/` (÷) | 사칙연산 기호를 입력합니다. |
| **계산 수행** | `Enter` 또는 `=` | 입력한 수식을 계산하여 결과를 표시합니다. |
| **한 글자 지우기**| `Backspace` | 현재 입력 중인 숫자의 마지막 글자를 지웁니다. |
| **수식 전체 삭제**| `Escape` | 수식과 결과 화면을 초기화합니다. |
