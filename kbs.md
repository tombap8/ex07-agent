# 디자인 규격서 (kbs.md)
# 나는 10년차 웹 디자이너다.

이 문서는 프로젝트의 핵심 디자인 가이드라인을 정의합니다. 다음 스타일 규격을 준수하여 화면을 구성합니다.

## 🎨 테마 및 색상 규격 (Color Palette)

계산기 웹 애플리케이션의 테마는 다음 색상 조합을 기준으로 디자인합니다:

| 요소 | 사양 | 색상 코드 (예시) |
| :--- | :--- | :--- |
| **배경 (Background)** | 오렌지(Orange) 색상 계열의 그라데이션 | `#ffedd5` ~ `#fed7aa` (라이트), `#431407` ~ `#1c1917` (다크) |
| **버튼 (Buttons)** | 밝은 회색(Light Gray) 그라데이션 배경 | `linear-gradient(#f4f4f5, #e4e4e7)` (기본/호버) |
| **테두리 (Borders)** | 차분한 회색(Gray) 테두리 | `#d6d3d1` (라이트), `#44403c` (다크) |
| **텍스트 (Text)** | 가독성이 높은 딥 오렌지(Deep Orange) / 차콜 색상 | `#7c2d12` / `#1c1917` (메인/강조) |

---

## 📐 스타일 적용 가이드

- **오렌지 배경 (Orange Background):** 전체 화면에 따뜻하고 감각적인 오렌지 톤의 그라데이션을 적용하여 프리미엄하고 에너지 넘치는 분위기를 조성합니다.
- **밝은 회색 그라데이션 버튼 (Light Gray Gradient Buttons):** 계산기의 조작 버튼들은 차분하고 금속 질감이 느껴지는 밝은 회색 그라데이션을 부여해 입체감과 세련미를 더합니다.
- **회색 테두리 (Gray Borders):** 계산기 요소들과 키패드 버튼 주위에 부드러운 회색 경계선을 배치해 깔끔한 레이아웃 가이드를 완성합니다.
- **딥 오렌지 텍스트 (Deep Orange Text):** 주요 결과와 조작 기호, 타이틀 텍스트에 딥 오렌지 또는 차콜 컬러를 매칭하여 눈의 피로를 최소화하면서 가독성을 확보합니다.
- **헤더 유틸리티 버튼 (Header Utility Buttons):** 상단 테마 전환 및 계산 기록 아이콘 버튼은 반투명 유리(Glassmorphism) 스타일을 독립 적용하고 내부 아이콘 색상이 배경색과 겹치지 않도록 대비를 대폭 올려 어두운 환경에서의 시인성을 확보하였습니다.
- **단축키 안내 배지 (Shortcut Badges):** 하단 키보드 단축키 안내 배지(Enter, Esc, Backspace 등)는 다크/라이트 모드 환경 각각에 맞춰 독립적인 키캡 배경색과 고대비 폰트 색상을 적용하여 텍스트 판독성을 대폭 개선했습니다.
