# 디자인 규격서 (kbs.md)

이 문서는 프로젝트의 핵심 디자인 가이드라인을 정의합니다. 다음 스타일 규격을 준수하여 화면을 구성합니다.

## 🎨 테마 및 색상 규격 (Color Palette)

계산기 웹 애플리케이션의 테마는 다음 색상 조합을 기준으로 디자인합니다:

| 요소 | 사양 | 색상 코드 (예시) |
| :--- | :--- | :--- |
| **배경 (Background)** | 민트(Mint) 색상 계열의 그라데이션 | `#a7f3d0` ~ `#34d399` (라이트), `#064e3b` ~ `#022c22` (다크) |
| **버튼 (Buttons)** | 선명한 노란색(Yellow) 배경 | `#fef08a` / `#facc15` (기본/호버) |
| **테두리 (Borders)** | 차분한 회색(Gray) 테두리 | `#cbd5e1` (라이트), `#475569` (다크) |
| **텍스트 (Text)** | 신뢰감을 주는 파란색(Blue) 색상 | `#1d4ed8` / `#1e3a8a` (메인/강조) |

---

## 📐 스타일 적용 가이드

- **민트 배경 (Mint Background):** 전체 화면의 톤을 지배하는 민트 색상은 눈의 피로를 덜고 산뜻한 느낌을 줍니다.
- **노란색 버튼 (Yellow Buttons):** 계산기의 숫자 및 기능 버튼들은 사용자 시선이 집중되도록 노란색 톤을 기반으로 구성하며, 호버/액티브 시 미세한 입체감을 줍니다.
- **회색 테두리 (Gray Borders):** 각 요소(계산기 본체 카드, 버튼, 디스플레이 영역 등)의 경계선은 연한 회색을 사용하여 깔끔하고 정돈된 그리드를 형성합니다.
- **파란색 텍스트 (Blue Text):** 가독성을 극대화하기 위해 흰색이나 검은색 대신 세련된 파란색 계열의 서체를 사용하여 현대적인 느낌을 제공합니다.
